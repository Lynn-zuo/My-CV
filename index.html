我的开源项目和作品

项目一：画一只皮卡丘

技术栈：HTML + CSS + jQuery

项目介绍：通过flex布局，利用CSS画一个皮卡丘，并将CSS样式代码动态打印出来。通过jQuery封装一个打印函数，结合setTimeout()动态打印CSS样式。

项目二：个人定制导航网页

技术栈：HTML + CSS + JavaScript

项目介绍：通过flex布局及原生js开发的个人导航网页，通过添加事件监听跳转对应按键绑定的网站，还可以自定义每个键绑定的导航网站。根据DOM事件委托机制取消事件冒泡，实现目标网站的修改，而阻止按键点击事件的发生。

项目三：移动端开心外卖Web App

技术栈：

前端-Vue + vue-router + vuex + ES6 + webpack + Vant

后端-Node + Express + Mongodb + Mongoose

项目介绍：基于外卖业务的前后台分离Web App，采用模块化、组件化、工程化的模式开发，具体涉及技术：1）通过ajax请求库axios封装ajax异步请求函数； 2）ES7最新语法: async/await； 3）利用vuex对组件的共享状态进行集中式管理状态； 4）使用代理服务器解决跨域问题； 5）mock数据接口库：mockjs； 6）一次性短信验证码和图片验证码技术等。

项目四：自律书圈社交小程序（开发中）

技术栈：WXML + WXSS + JavaScript + WXS

项目介绍：项目封装了实现局部页面内路由切换功能的自定义Tabs组件，可以自由添加切换项数量；通过各种微信前端请求接口和云函数、云数据库等云开发技术实现了微信小程序登录及点赞功能，开发读书动态笔记分享社交小程序。

项目五：微信云课堂小程序（开发中）

技术栈：WXML + WXSS + JavaScript + WXS + Vant-weapp

项目介绍：为了熟悉小程序开发流程，基于云课堂业务开发的小程序，利用组件库实现了首页的查询、轮播及宫格导航等界面效果。
